# 烹饪步骤
1. 豆腐压碎 ==> 导入碗中，加入几个鸡蛋 ==> 加少许盐，充分搅拌均匀
2. 放入蒸笼，上汽后蒸10分钟
3. 空碗里加入以下材料，调出均匀料汁备用：
   - 黄豆酱，少许
   - 生抽，半勺
   - 蚝油，少许
   - 老抽，少量，调色用
   - 鸡精
   - 花椒面
   - 水淀粉，大半勺
4. 锅内油热，下入猪肉末，**小火**慢炒，把肉末炒散炒熟 
5. 加入姜末和葱末，翻炒均匀，炒出香味
6. 倒入料汁，**大火**熬至浓稠
7. 倒出浇在蒸好的豆腐上
