
# 食材
- 鸡翅
- 可乐
- 柠檬。切片
- 葱，切片
- 姜，切片

# 步骤
1. 鸡翅去血水
	   1. 在鸡翅两面用刀划几道
	   2. 把鸡翅放入清水中，用水抓揉几下，浸泡**半小时**
2. 腌制鸡翅
	   1. 用厨房纸吸干鸡翅表面水分，否则影响腌制入味
	   2. 讲大块切好的葱 + 姜加入鸡翅
	   3. 加入半瓶盖的花雕酒
	   4. 少量生抽和老抽
	   5. 适量盐
	   6. 抓拌1~2分钟，放置 10 分钟
3. 煎鸡翅
	   1. 无油锅内，把鸡翅外面朝下，均匀平铺在锅内
	   2. 开大火，不要动鸡翅，把鸡表面的油脂煎出来
	   3. 这时候稍微晃动锅，让鸡翅在锅面滑动，鸡皮就不会脱了
	   4. 一面煎至金黄，再翻面煎至金黄，在此过程中可以加入腌鸡翅用的葱和姜
	   5. 当晃动锅，鸡翅能够转起来，说明鸡翅已经煎至金黄
4. 炖鸡翅
	   1. 分次加入可乐，少量多次加会避免快速降温，鸡翅就不会煮成又老又柴。
	   2. 可乐加到与鸡翅刚刚几乎持平。
	   3. 烧开后，加入柠檬片（碱性，起软化肉质，且增香）
	   4. 炖 10 分钟，鸡翅两边的骨头都伸出来了
	   5. 根据喜好，淋入适量老抽，开大火收汁，收汁过程中挑出葱姜
	   6. 当汤汁粘稠并裹上鸡翅，撒入小葱和白芝麻
	   7. 出锅
