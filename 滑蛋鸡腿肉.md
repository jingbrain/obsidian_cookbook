
# 食材
- 小鸡腿，4个
- 鸡蛋，2个
- 洋葱，切丝

# 做法
1. 小鸡腿去骨去皮切块，加入
	   - 料酒，一勺
	   - 生抽，一勺
	   - 淀粉，一勺
	   - 黑胡椒
	抓拌腌制 **20 分钟**
2. 两个鸡蛋，打散备用
3. 制作酱汁
	   - 生抽，2勺
	   - 蚝油，1勺
	   - 白糖，半勺
	   - 黑胡椒
	   - 清水，半碗
4. 锅内油热放入*鸡块*，煎至**两面焦香**，盛出备用
5. 锅内油热炒香*洋葱丝*
6. 平铺放入*鸡块*，淋入*酱汁*，盖盖**小火**焖煮 **10 分钟**
7. 开盖淋上*鸡蛋液*，凝固后撒上*芝麻*和*葱花*
