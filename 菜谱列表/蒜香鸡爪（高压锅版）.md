#📌家常置顶 
# 食材
- 鸡爪
- 蒜末
- 姜片
- 小米辣
- 大料（八角，桂皮）
- 干辣椒
- 猪油

# 步骤
1. 鸡爪减去指甲，洗干净，加*姜片+葱+料酒***焯水**捞出
2. 鸡爪放入高压锅，加入*八角 + 桂皮 + 姜片 + 干辣椒*
3. 倒入适量*啤酒 + 盐 + 生抽 +老抽 + 蚝油 + 胡椒粉 + 少量糖*，液面淹没鸡爪一半高度
4. 高压锅**上汽后压4 分钟**，关火，捞出大料
5. 铁锅内热锅加入*猪油*
6. 加入 *（大量）蒜末 + 小米辣末 + 辣椒面 + 盐 + 生抽 + 鸡精*，开小火炒香
7. 打开高压锅，捞出大料 ==> 倒入上面炒料，搅拌均匀 ==> 煮开，稍微收汁 ==> 撒上*葱花*，搅拌均匀，出锅
