# 食材清单

- 白菜，适量
- 酱油 : 醋 : 糖 = 1 : 3 : 2
- 葱，少许
- 蒜，1瓣，顺丝切易出味
- 辣椒，适量
- 淀粉，适量
- 油，适量

<aside> 💡 酱油 : 醋 : 糖 = 1 : 3 : 2

</aside>

# 操作步骤

1. **备菜**：白菜取帮，斜刀片切，再从中间切开两半。
2. **调汁**：生抽 : 米醋 : 糖 = 1 : 3 : 2 比例调好，少来一点清水，放入**葱花**和**蒜片**，再放适量**淀粉**。
3. **白菜焯水**：水烧开加两滴油，烫一下切好的白菜。不必等水再烧开，断生即可盛出，沥水备用。
4. **锅炒**：油热，大火，放入白菜，干辣椒（不吃辣不放），调好的糖醋汁。稍炒盛出即可。
