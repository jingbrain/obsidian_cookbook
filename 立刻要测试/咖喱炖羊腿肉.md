
# 食材准备

- 羊腿肉块（带骨）
- 咖喱膏（约 2-3 大勺，根据口味调整）
- 土豆（切块）
- 胡萝卜（切块）
- 洋葱（切丝或切块）
- 姜片、蒜瓣
- 椰浆或清水（约 400-500ml）
- 盐、糖、胡椒粉（适量）
- 食用油

# 步骤

1. 煸香底料
    - 热锅加油，放入*洋葱*炒至**微微金黄**。
    - 加入*姜片*和*蒜瓣*，炒香。
2. 煎羊肉
    - 将*羊腿肉块*放入锅中，煎至表面**微微焦黄**，锁住肉汁。
3. 加入咖喱膏
    - 放入*咖喱膏*，翻炒均匀，让羊肉和洋葱充分裹上咖喱香料。
    - 想要辣味，可以在此时加入*辣椒*或*辣椒粉*。
4. 加液体炖煮
    - 倒入*椰浆*或*清水*（没过食材），**大火煮开后转小火慢炖**。
    - 如果喜欢更浓郁，可以加少量*黄油*。
    - 炖约 **40-60 分钟**，直到羊肉变软。炖煮时间越长，羊肉越软烂，风味更好。
5. 加入蔬菜
    - 放入*土豆*和*胡萝卜*，继续炖 **20 分钟**，直到蔬菜熟透。
6. 调味收汁
    - 根据口味加*盐*、*糖*、*胡椒粉*。
    - 如果汤汁太多，可以稍微开大火收汁，保持浓稠度。
